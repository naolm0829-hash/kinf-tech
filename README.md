# ክንፍTech (KinfTech)

Ethiopian EdTech platform empowering children with Autism, Dyslexia, and ADHD through
AI-powered learning, visual communication boards, and bilingual (Amharic/English)
content.

## Tech Stack

- React 18 + Vite + TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase (auth, database, edge functions)

## Local development

```bash
bun install
bun run dev
```
